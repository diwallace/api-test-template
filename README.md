# Projeto de template para testes de API

## Dependências
* JDK 14.0
* Gradle 6.4
* TestNG 7.1.0
* RestAssured 4.3.0
* Gson 2.8.6

## Rodando o projeto
`gradle clean test`
<pre>
Após rodar o projeto é gerada uma pasta build e report de execução fica no seguinte caminho build>reports>tests>index.html
</pre>
